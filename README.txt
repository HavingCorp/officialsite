HAVING - 홈 CTA 줄바꿈 + 애널리틱스/메일 연결
==================================================

[바뀐 파일 4개 - 덮어쓰기]
  index.html      홈 하단 문의 CTA 제목 줄바꿈
  lang/ko.json    CTA 국문 문구 (줄바꿈 포함)
  lang/en.json    CTA 영문 문구 (줄바꿈 포함)
  config.js       애널리틱스 측정 ID + 문의 메일 키


1) 홈 CTA 제목 두 줄로
--------------------------------
  국문:  성공적인 K-뷰티 비즈니스,
         해빙과 시작하세요.
  영문:  Let's find the right
         K-beauty opportunity.

  제목이 i18n(data-i18n-html)으로 렌더되므로
  lang 파일의 문구에 <br/> 을 넣어 처리했습니다.


2) config.js (앞서 작업분 포함)
--------------------------------
  gaMeasurementId    : G-DJ3QNYES9V   구글 애널리틱스
  web3formsAccessKey : 86192900-...    문의 메일 새 키

  * 이 저장소(zip)엔 옛 값들이 남아 있어 함께 반영했습니다.
  * GA는 쿠키 동의 배너에서 [동의] 시에만 로드됩니다.


[검증]
  · CTA 국문/영문 모두 2줄 렌더 확인 (br 1개)
  · JS 에러 없음
  · 애널리틱스 동의 게이트 정상 (동의 전 로드 안 됨 / 동의 시 로드)


[참고]
about.html 등 다른 페이지는 이 패키지에 없습니다.
어바웃 수정(실적 삭제, 지구본)과 SEO 패키지는 별도로 이미
전달/배포된 상태입니다. 이번 건은 홈 CTA와 config.js 뿐입니다.
