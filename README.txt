=====================================

  js/finished.js      브랜드 모달 / 로고 게이트 / 확장자 자동탐색
  css/finished.css    모달 스타일 (기존 파일 끝에 추가)

==========================================
!! 로고가 있어야 브랜드가 보입니다 !!
==========================================
  로고 있음  ->  브랜드 카드 표시
  로고 없음  ->  브랜드 카드 자체가 안 나옴

13개 브랜드 모두 대기 상태입니다.
로고를 올리는 순간 해당 브랜드가 자동으로 등장합니다.
따로 JSON을 고칠 필요 없습니다.

[주의] 로고를 하나도 안 올리면 완제 페이지가 비어 보입니다.
       최소 1~2개는 올린 뒤 배포하세요.

OEM / ODM 카드:  brands.json 맨 위 "show_oem": false (현재 숨김)
                 표시하려면 true 로 변경

특정 브랜드를 로고가 있어도 감추고 싶으면:
  brands.json 에서 그 브랜드의 "published": false

==========================================
이미지 (전부 assets/brands/ 한 폴더)
==========================================
1) 로고  {id}-logo.png      ** 필수 - 이게 있어야 브랜드가 나옴 **
   800 x 400 투명 PNG / 안전영역 720x320 / 좌측·세로중앙

2) 히어로 {id}-hero.jpg     (선택)
   1600 x 900 / 하단 40%에 슬로건이 올라감 (중요 요소는 위 2/3)
   없으면 브랜드 색 그라데이션으로 자동 대체

3) 제품컷 {id}-01.jpg ~ -06.jpg   (선택, 최대 6장)
   900 x 900 흰 배경 / 안전영역 720x720 / 가운데
   없으면 사진 영역 자체가 안 보임

확장자는 jpg/png/webp 자동 인식. 단, 로고는 투명이 필요하므로 PNG.
용량 권장: 히어로 200~400KB / 제품컷 100~200KB

==========================================
자동 변환 스크립트 (이미지변환/)
==========================================
pip install pillow
로고 :  python3 logo_normalize.py raw out
제품컷: python3 logo_normalize.py sku_raw sku_out --sku

여백 트림 -> 안전영역 스케일 -> 캔버스 배치 자동 처리
