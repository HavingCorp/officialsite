HAVING 업데이트
=====================================

[바뀐 파일 4개 - 덮어쓰기]
  ingredients.html    원료 오탈자 수정 (ㄴ -> LEKWhite AGS)
  js/finished.js      브랜드 모달 / 로고 게이트 / SKU명 숨김 / 배경 스크롤 잠금
  css/finished.css    모달 스타일
  data/brands.json    13개 브랜드

이번 수정 내용
--------------
1. 원료 페이지 5번째 항목 이름이 'ㄴ' 으로 들어가 있던 것을
   'LEKWhite AGS' 로 수정했습니다.
   ** 원본 엑셀(원료(국문) 시트 5행 제품명)도 같이 고쳐두세요.
      안 그러면 나중에 데이터 재생성 시 다시 들어갑니다.

2. 브랜드 모달에서 SKU 이름을 표시하지 않습니다.
   (사진 순서와 이름이 어긋나는 문제)
   사진만 보여주고, 클릭하면 히어로만 바뀝니다.

3. 모달이 열려 있을 때 뒤쪽 배경이 스크롤되지 않습니다.
   닫으면 원래 보던 위치로 정확히 돌아옵니다.


==========================================
로고가 있어야 브랜드가 보입니다
==========================================
  로고 있음  ->  브랜드 카드 표시
  로고 없음  ->  카드 자체가 안 나옴

13개 전부 대기 상태. 로고를 올리면 자동으로 등장합니다.
[주의] 로고를 하나도 안 올리면 완제 페이지가 비어 보입니다.

OEM / ODM 카드: brands.json 맨 위 "show_oem": false (현재 숨김)


==========================================
이미지 규격 (전부 assets/brands/)
==========================================
1) 로고  {id}-logo.png    ** 필수 **
   800 x 400 투명 PNG / 안전영역 720x320 / 좌측·세로중앙

2) 히어로 {id}-hero.jpg   (선택)
   1600 x 900 / 하단 40%에 슬로건이 올라감
   없으면 브랜드 색 그라데이션으로 대체

3) 제품컷 {id}-01.jpg ~ -06.jpg  (선택, 최대 6장)
   900 x 900 흰 배경 / 안전영역 720x720 / 가운데
   * 순서는 자유입니다. 이름을 표시하지 않으므로 맞출 필요 없습니다.

확장자 jpg/png/webp 자동 인식 (로고만 투명 PNG 필요)


==========================================
자동 변환 (이미지변환/)
==========================================
pip install pillow
로고 :  python3 logo_normalize.py raw out
제품컷: python3 logo_normalize.py sku_raw sku_out --sku
